Title of the project:

Multi Authority Attribute Based keyword Search Over Encrypted Data

Purpose:

This application is present a secure Multi-Authority CP-ABKS(MABKS) system to address such limitations and minimize the computation and storage budden on resource-limited devices in cloud system.

Abstract:

  • Searchable Encryption (SE) is an important technique to guarantee data security and usuability in the cloud at same time

  • Ciphertext-policy Attribute-Based Encryption,the Ciphertext-policy Attribute-Based Keyword Search scheme can achieve keyword based retrival and fine-grained access control simultaneously.

  • However the single attribute authority in existing CP-ABKS scheme is tasked with costly user certificate verification and secret key distrubtion.

  • In addition,this result in a single point performance bottleneck in distrubted cloud systems.
